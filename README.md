# 💱 Currency Converter

A modern and responsive **Currency Converter** web application built with **HTML, CSS, and JavaScript**. This project enables users to convert amounts between different currencies using real-time exchange rates through an external API.

Designed with a clean user interface and seamless user experience, the application provides fast and accurate currency conversions across multiple global currencies.

---

## ✨ Features

- 🌍 Real-time currency conversion
- 🔄 Exchange rates fetched dynamically via API
- 💱 Support for multiple international currencies
- 🔁 One-click currency swap functionality
- 📱 Fully responsive design for mobile, tablet, and desktop devices
- ⚡ Instant conversion results
- ✅ Input validation for improved user experience

---

## 🛠️ Tech Stack

- **HTML5** – Semantic structure and layout
- **CSS3** – Responsive styling and user interface design
- **JavaScript (ES6+)** – Application logic and API integration
